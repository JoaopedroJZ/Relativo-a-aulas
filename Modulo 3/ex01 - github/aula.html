Git/Github

o que é Git?

O Respositório é um local onde você guarda coisas, la também você pode fazer backup de seus arquivos, os sites que você criou até agora foi no seu computador, todos os sites estão dentro do seu computador (repositório LOCAL), para gerenciar backups você pode usar um GERENCIADOR de VERSÕES que serve para armazenar diversas versões do seu site, por que se caso você não esteja satisfeito com o trabalho atual você retrocede a versão anterior e continua de lá, o nome desse software é GIT.

O que é Github?

Vamos dizer que você queira mostrar um site para um amigo, para isso você precisa hospedar o site em um servidor, para que assim o servidor envie os arquivos do site para seu amigo, o lugar que você vai poder hospedar gratuitamente seu site é o Github, porém saiba que o Github é somente para iniciantes, ele não é recomendado para projetos maiores.

Exemplos

Você tem sua pasta com os arquivos do site e o Git guardando todas as versões dela você vai chamar a pasta com o Git de Repositório Local, o servidor ao invés de ser chamado de GitHub você vai chamar ele de Repositório Remoto, isso é bom para trabalho em equipe, cada um vai ter seu repositório local e atualizar do seu jeito e ele vão poder enviar para o repositório remoto.

Primeiro Repositório.

para criar um repositório basta você ir no "github desktop" e clicar em "criar novo repositório" e então você coloca o EXATO nome da pasta dos seus arquivos, e em "local path" você coloca um pasta antes da pasta que você quer que seja selecionada.

Configurações de Background

Baixando Imagens

Vá nos Repositório do GitHub e baixe as imagens da aula/capitulo, para isso entre no repositório do "Curso em Vídeo" e clique no capitulo que corresponde essa aula e então baixe o arquivo de imagens!

Colocando Imagem de Fundo

Para utilizar imagens de fundo você deve usar a tag "Background-Image: url" e colocar a imagem dentro dos parenteses no URL. A imagem vai se repetir até cobrir TODO o local que você pediu, se caso a imagem for muito grande pro espaço que você quer colocar, ela vai mostrar somente uma parte, para você evitar que uma imagem fique se repetindo ou se repetir de um jeito mais especifico use a tag "Background-Repeat" nas seguintes classificações:

  • Repeat
  • No-Repeat
  • Repeat-X
  • Repeat-Y

Background-Size

Com a tag "Background-image" você aumenta e diminui as imagens de fundo sendo a primeira classificação a altura e a segunda largura, você pode usar o "COVER" para ela cobrir TUDO, a "CONTAIN" vai manter a imagem sem NENHUM corte.

Inline-Block

a tag "display: inline-block", serve para você colocar uma DIV em INLINE mas que ela continue sendo afetada pelas configurações que um BLOCK seria

Background-Position

Por padrão toda a imagem começa sendo mostrado do canto superior esquerdo, você pode mudar isso se utilizando do "Background-Position", com essa tag você pode mudar o lugar onde ela começa a ser mostrada sendo canto inferior/superior e esquerdo/direito, você também colocar ela centralizada os comandos são:

  • Left Top
  • Left Center
  • Left Bottom
  • Center Bottom
  • Right Bottom
  • Right Center
  • Right Top
  • Center Top
  • Center Center

Background-Attachment

com a tag "Background-attachment" com a especificação "fixed" você pode fixar a imagem em um fundo que não se mexe e que sempre se mantenha na mesma posição da imagem, você pode usar um shorthand de background, ela usa na seguinte ordem:

  1. Color
  2. Image
  3. Position
  4. Repeat
  5. Size
  6. Attachment

Obs:O background-size não é aceitado na shorthand, então ela fica na mesma ordem só que sem o size.

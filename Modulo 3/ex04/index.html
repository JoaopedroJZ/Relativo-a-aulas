Tabelas HTML

Para criar uma tabela crie uma tag chamada "table", uma coisa importante antes de tudo é saber que todas as tabelas tem um hierarquia simples, a hierarquia de tabela simples é:

  • TABLE = TABELA
  • TABLE ROW "TR" = LINHA DE TABELA
    (DENTRO DA TABLE)
  • TABLE HEADER = CABEÇALHO DE TABELA (DENTRO DO TABLE ROW)
  • TABLE DATA "TD" = DADO DE TABELA

Borda de Tabela

Coloque a tag "border" na tag de TD (table data), e então você vai ter uma "tabela", porém é recomendado você colocar na tag "table" outra tag chamada de "border-collapse" na classificação de "collapse", assim ela vai ficar juntinha como em uma tabela.

Classificação das linhas

Você pode classificar os TR (TABLE ROW) e os TD (TABLE DATA), para destacar informações e areas especificas da tabela.

Alinhamento de Tabelas

Para alinhar tabelas você pode utilizar "Text-allign" e especificar alguma posição isso é seu alinhamento orizontal, então como fazer um alinhamento horizontal? use "vertical-allign" o padrão do vertical allign é MIDDLE, diferente do text allign que é LEFT ou CENTER.

Tabelas Grandes

A anatomia de uma tabela grande é basicamente, THEAD, TBODY e TFOOT. area da cabeça, area do corpo e area do rodapé, dentro de todas essas tags você pode usar também TD, TR e TH. A estrutura da tabela é algo extremamente importante de seguir, normalmente no thead é utilizado TH's para organizar melhor afinal algo que esta no cabeçario normalmente não é um dado e sim uma informação. DICA: Caso for fazer alguma tabela pequena continue fazendo ela com thead, tbody e tfoot, pois assim você treina sua semântica.

"Legenda da Tabela"

Caption nada mais é que uma legenda, ela vem antes do thead bem abaixo do table, você não pode colocar tr, td e th somente texto.

Scope

Scope é aquilo que contextualiza onde/oque é as tags, se o TITULO esta do LADO ou em CIMA dos DADOS, sendo em cima "Scope=col" e do lado "Scope=row", isso serve para deixar tudo mais SEMANTICO. Lmebrando que quanto mais semantico mais recomendado o seu site será.

Efeito Zebrado

O efeito zebrado é normalemnte utilizado para tabelas grandes onde o cerebro humano não consegue localizar completamente um dado, para fazer um efeito zebrado basta você colocar uma cor no "tbody > tr", porém só isso não basta e é por isso que você vai utilizar uma pseudoclasse chamada nth-child(2n), essa tag faz com que você a cada 1 linha deixe o tr com o background color que você especificou, você também pode fazer isso em outras coisas porém a maioria delas você apenas utiliza em tabelas. e é assim que se gera um efeito zebrado.
NOTA: Você também pode colocar o efeito zebrado aparecer a cada 3 ou mais linhas.

Cabeçalho Fixo

Fique ciente que este recurso não funciona em TODOS os navegadores e também que esse efeito SÓ serve se caso a tabela for GRANDE. Para começar sua TABLE precisa ter um position RELATIVO e então você da um caminho de "thead > tr > th" e coloca sua position STCIKY e coloque seu TOP em ZERO (0) e assim basta colocar seu background color padrão e PRONTO.

Mesclagem de Celulas

Para fazer uma mesclagem de celulas você deve saber a base de tabelas, mesclagens de celulas nada mais é que uma celula ocupar o lugar de duas. porque isso? simples, para não ficar um espaço em branco em uma tabela. COMO FAZER? você coloca ao lado do td/th "colspan=2", assim ele vai ocupar DUAS COLUNAS, você pode mudar o numero para ela ocupar mais ou menos colunas. você também pode fazer uma coluna expandir e ocupar mais LINHAS, para isso basta você colocar "rowspan=2" e assim ele vai ocupar 2 LINHAS.

Scopes em Grupos

O "colgroup" ou "rowgroup" são utilizados quando um TH ocupa mais de um span, no caso se você tem um titulo pra definir 3 dados na mesma linha você utiliza esse código.

Agrupamento de Colunas/Linhas

Se é pra modificar uma linha especifica é muito simples, porém o problema vem quando é uma COLUNA, você poderia ir em cada uma delas colocando uma class especifica, mas e se sua tabela tiver 500 linhas? pra fazer isso de maneira rapida basta você criar uma tag chamada "colgroup" a qual você cria dentro da tabela, la você coloca a tag COL e cada uma delas vai automaticamente vincular com as scopes que você colocou no TH EM ORDEM DE PRIMEIRO E ULTIMO A SER COLOCADO NO CÓDIGO, para separar cada uma dos cols basta você coloca uma class em cada col que você colocou dentro do colgroup, você também pode utilizar a regra do SPAN para aplicar a mesma class em duas ou mais colunas neste caso você coloca apenas a primeira col de coluna e segunda que seria a qual você quer colocar a mesma configuração você não precisa colocar. Simples facil e rápido
NOTA: O mesmo vale para as linhas porém nesse caso você usa "rowgroup"

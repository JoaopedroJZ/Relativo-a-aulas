Tabelas HTML

Para criar uma tabela crie uma tag chamada "table", uma coisa importante antes de tudo é saber que todas as tabelas tem um hierarquia simples, a hierarquia de tabela simples é:

  • TABLE = TABELA
  • TABLE ROW "TR" = LINHA DE TABELA
    (DENTRO DA TABLE)
  • TABLE HEADER = CABEÇALHO DE TABELA (DENTRO DO TABLE ROW)
  • TABLE DATA "TD" = DADO DE TABELA

Borda de Tabela

Coloque a tag "border" na tag de TD (table data), e então você vai ter uma "tabela", porém é recomendado você colocar na tag "table" outra tag chamada de "border-collapse" na classificação de "collapse", assim ela vai ficar juntinha como em uma tabela.

Classificação das linhas

Você pode classificar os TR (TABLE ROW) e os TD (TABLE DATA), para destacar informações e areas especificas da tabela.

Alinhamento de Tabelas

Para alinhar tabelas você pode utilizar "Text-allign" e especificar alguma posição isso é seu alinhamento orizontal, então como fazer um alinhamento horizontal? use "vertical-allign" o padrão do vertical allign é MIDDLE, diferente do text allign que é LEFT ou CENTER.

Tabelas Grandes

A anatomia de uma tabela grande é basicamente, THEAD, TBODY e TFOOT. area da cabeça, area do corpo e area do rodapé, dentro de todas essas tags você pode usar também TD, TR e TH. A estrutura da tabela é algo extremamente importante de seguir, normalmente no thead é utilizado TH's para organizar melhor afinal algo que esta no cabeçario normalmente não é um dado e sim uma informação. DICA: Caso for fazer alguma tabela pequena continue fazendo ela com thead, tbody e tfoot, pois assim você treina sua semântica.

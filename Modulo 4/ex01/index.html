Iframes

Iframes

O que é iframe? o "i" do iframe significa inline, ou seja um frame inline/quadro em linha. O iframe serve para criar uma janela/aba dentro do seu site para mostrar outros sites e videos, apareceram duas especificações, uma de frameborder que quando maior o numero maior a borda e o outro é a src que é o link do site/video que você quer colocar.

Ajustando Iframes

Uma coisa importante a se fazer é colocar um link no iframe para se caso haja incompatibilidade apareça o link para assim o leitor não perder nenhum conteudo, você pode ajustar altura e largura, fique ciente que o tamanho padrão é de 300x150 pixeis de tamanho. Nas iframes as css's são soberanas ou seja elas são preferenciais comparadas a css's na própria tag. Você pode configurar um "scolling" que é a rolagem de um iframe que tem as configurações "auto" que é a automatica, "yes" que mesmo que ele ja caiba dentro da caixa ele vai aparecer mesmo assim e por fim "no" que para não ter rolagem. Frame border tem somente 2 especificações que no caso é "0" e "1", para personalizar a borda basta tirar o frame border e colocar a border no css.

Iframe Estatico

Quando você entra na sua pagina o seu iframe esta normalmente em branco, o ideal seria ele aparecer algo entende? pra isso você pode colocar a primeira opção de pagina no src, você também pode colocar um srcdoc, la você fazer um HTML inteiro dentro do iframe. esse mesmo src ou srcdoc só vai aparecer uma unica vez.

Incoscistencias dos Iframes

Os iframes tem algumas incoveniencias, pois algumas vezes o GOOGLEBOT que é o bot que analiza os sites que vão estar no topo tem problemas para identificar iframes e ele acaba não recomendando o seu site, ou seja, conteudos importantes não são recomendados serem colocados em iframes por conta disso. Pessoas cegas conseguem usar o computador graças a um software de leitor de tela, o problema é que esses software podem não identificar o seu iframe e assim alguém cego não pode ter 100% de aproveitamento do seu site.

Tornando Iframes mais Seguros

é comum que ataques hackers capturem cadastros de usarios um jeito de proteger mais isso é colocar o login em um site SEPARADO e colocar um iframe no site principal linkando o site de login/cadastro, você também deve colocar uma especificação no iframe chamada SANDBOX="SANDBOX" com isso você cria uma proteção maior pois ele não está mais capturando dados do usuario. Outra coisa que você colocar no iframe é o referrerpolicy="no-referrer", isso deixa o iframe mais limitado. Mas vamos imaginar que no seu site você quer que o usario preencha o formulario, para isso vá no sandbox="allow-same-origin" que permite coisas da mesma origem, você pode dar um espaço e também colocar dentro do sandbox "allow-forms" e "allow-scripts" assim você vai ter mais possibilidades no iframe e mantendo o mesmo nivel de segurança. Futuramente teremos JavaScripts, para bloquear um script no iframe é só você colocar novamente sandbox="sandbox", se você quiser habilitar SOMENTE scripts coloque "allow-scripts".

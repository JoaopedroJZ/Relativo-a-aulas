Formularios

1. O que é um Formulario?

Um formulario nada mais é que um lugar onde o usario faz um cadastro ou um login, os formularios podem ser preenchidos tanto de forma fisica quanto de forma digital.

2. Como fazer um Formulario?

Todo o formulario deve estar dentro de uma tag "FORM", para fazer as "caixinhas" onde você preenche com nome, senha, data de nascimento e etc, basta você usar uma tag chama "INPUT", o input tem MUITAS definições dentre elas para o NOME você utiliza o com a forma de "INPUT type TEXT" dentro dessa tag terá duas especificações, NAME e ID, lá você coloca oque você quer cobrar do usario seja NOME ou SOBRENOME. Como desligar o auto preenchimento do input? vá na tag "form" e coloque "autocomplete: off".

3. Label, Name e Id

O seu navegador é burro, ele não sabe diferenciar um SOBRENOME de um NOME pra isso utilizamos LABEL que é um identificador, em linguagens de programação isso é importante pois quando você ENVIA o dado ele vai ser identificado por Label, Name e Id. Sendo em PHP mais comum o Name e em javascript o ID. O label é basicamente uma etiqueta que vai identificar oque é oque dentro do seu formulario, sempre antes do input você vai colocar algo pra pessoa identificar oque ela vai colocar, utilizamos o label para o COMPUTADOR identificar que X texto é pra Y input e vice-verse, dentro da tag label tem uma especificação que é o "for" no for você coloca o ID ou o NAME, a importancia disso é identificar e facilitar para os navegadores.

4. Pra onde é enviado as informações?

Normalmente se manda os dados para uma linguagem de programação sendo as duas mais comuns Java e PHP (backend), pra isso coloque um "action" no seu form no action você coloca o arquivo da linguagem que você estiver usando.

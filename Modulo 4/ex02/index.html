Formularios

1. O que é um Formulario?

Um formulario nada mais é que um lugar onde o usario faz um cadastro ou um login, os formularios podem ser preenchidos tanto de forma fisica quanto de forma digital.

2. Como fazer um Formulario?

Todo o formulario deve estar dentro de uma tag "FORM", para fazer as "caixinhas" onde você preenche com nome, senha, data de nascimento e etc, basta você usar uma tag chama "INPUT", o input tem MUITAS definições dentre elas para o NOME você utiliza o com a forma de "INPUT type TEXT" dentro dessa tag terá duas especifações, NAME e ID, lá você coloca oque você quer cobrar do usario seja NOME ou SOBRENOME.

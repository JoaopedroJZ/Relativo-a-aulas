Tipos de Alinhamentos

existem 3 tipos de alinhamentos, CENTRO, DIREITA e ESQUERDA. antigamente para colocar um texto centralizado bastava utilizar tag "CENTER", porém essa é uma tag obsoleta. o jeito certo é ir no CSS e colocar a tag "Text-allign". outro tag de posição é o "text-indent", que é basicamente o espaço que você da ao escrever um "paragrafo".


Seletores Class/ID

o CLASS e ID servem para diferenciar um h2 de outro h2 por exemplo, você quer um h2 vermelho e outro h2 verde, para isso você coloca uma class pra cada um então você vai no CSS e coloca .nomedaclass e coloca suas especificações, o ID é a mesma coisa mas ao invés de usar .nomedoID use "tagutilizada#nomedoID"

  • tudo que em HTML é ID vira #
  • tudo que em HTML é CLASS vira .

a diferença entre CLASS e ID é que o ID só pode ser usado em uma unica tag, o class você pode usar em diversas tags, você pode usar o CLASS e ID na mesma tag

Personalizando Links

temos algumas pseudo classes nos links, que são visited e non-visited que no caso é que se caso você tenha visitado o link ele vai ficar de tal tipo e se caso não tenha visitado o link ele vai ficar diferente, para tirar o sublinhado do link basta colocar "text-decoration none", o pseudoclasse "ACTIVE" é quando você pressiona alguma coisa com seu mouse, o "after" siginifica depois do elemento e "before" significa antes do elemento, qual a diferença entre pseudoclasse e pseudoelemento? eles mexem diretamente no conteudo (ou seja qualquer coisa)

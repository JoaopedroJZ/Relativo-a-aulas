tags de css e html

tag Main

você pode criar uma tag para simular algo principal que no caso é a tag "main", basicamente uma tag de div só que ela sinaliza que é o conteudo principal

tags box shadow, border radius, margin e padding

a tag "box shadow" e "text shadow" servem para criar sombra nos textos ou formas, os 3 primeiros parametros do shadow é:

  1. o quanto essa sombra vai ir para o lado
  2. o quanto essa sombra vai para baixo
  3. o quanto essa sombra vai espalhar
  4. e por fim a cor

agora a tag "border radius" é para deixar esse "redondinho" no quadrado de texto que você ta vendo, você também pode configurar o tamanho

a tag "margin" serve para centralizar a caixa branca a qual você esta vendo (na configuração auto)

a tag "padding" serve para ajustar o texto de forma que ele não fique todo encolhido dentro da caixa

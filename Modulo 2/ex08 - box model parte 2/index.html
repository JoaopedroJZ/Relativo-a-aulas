outras informações

existem 4 margins, a margin top, right, bottom e left, ao invés de configurar 4 tags de cada margin utilize o shortcut e colocar as 4 margins, isso na ordem de:

  1. top
  2. right
  3. bottom
  4. left

para deixar um elemento no meio, basta utilizar margin no AUTO. você pode criar uma OUTLINE que é algo que fica entre a MARGIN e a BORDER. para tudo existem shortcuts, isso também é o caso das BORDERS e OUTLINES, que no caso uma BORDER é width, style e color. para PADDING é top, right, bottom e left. OUTLINE, width, style e color.

tags semanticas

você pode criar tags semanticas, que no caso são tags que são basicamente divs sóq semanticas que no caso são header, divs, nav, section, footer e article.

  • header = cabeçario
  • div = divisão/sessão
  • nav = area de navegação/menu
  • section = sessão
  • footer = créditos/desenvolvido por
  • article = assunto/noticia

Sombras

uma coisa importante é que você não precisa exagerar nas sombras, você pode ir ao devtools/inspecionar e criar a sombra por la e depois basta copiar e colar a tag. a tag box shadow segue a seguinte ordem:

  1. espaçamento pro lado
  2. espaçamento pra baixo
  3. blur (pode não ser utilizado)
  4. espalhamento

para colocar sombra em letras basta colocar text-shadow, você também pode fazer uma fonte INTERNA, basta colocar INSET no box shadow.

Arredondamento de Caixas

a Tag utilizada para arredondar as "caixas" é border-radius, assim como as outras tags o border radius tem 4 especificações, top, right, bottom e left. é bom lembrar que também é possivel se utilizar de porcentagem no border radius.

Bordas Personalizadas

você pode fazer bordas em programas como photoshop, gimp e qualquer programa de edição de imagem, para isso você deve criar um arquivo 80x80, depois de ter feito sua borda coloque o arquivo da borda no servidor, depois coloque uma tag de "border-image-scource", depois uma de "border-image-slice" e por fim "border-image-repeat", o shorthand dessas tags é "border-image". também é necessario uma configuração da tag "border" para se utilizar da borda personalizada.

NEGRITO / DESTAQUE

para utilizar o negrito basta usar a tag strong, desse jeito(semantico)
ou você poder usar a tag b (não semantica) olá
ou seja, ao invés de usar B use STRONG

ITALICO / ENFASE

você pode usar algo em italico com i (não semantico)
para usar utilizar enfasê você pode usar em(semantico)
ou seja ao invés de usar i use EM

PRINCIPAIS FORMATAÇÕES 2

TEXTO MARCADO

nós podemos marcar um texto com a tag mark(não existe mais OBSOLETA),
se caso você queira mudar a cor da marcação utilize a tag STYLE junto da tag BACKGROUND-COLOR
outro jeito é indo la na HEAD do código e colocar STYLE e dentro da STYLE coloque MARK, assim todas as MARKS vão ficar de uma cor especifica

TEXTO GRANDE E PEQUENO

você pode usar BIG e SMALL para deixar a letra grande e pequena (oláaa oláaaa), elas provavelmente vão se tornar obsoletas em breve

TEXTO DELETADO

podemos manter um texto como excluido (deve ser lido mas não considerado) basta utilizar a tag DEL, você é gay

TEXTO INSERIDO

podemos marcar um texto inserido para dar enfase ao texto com a tag INS, você é homi

TEXTO SOBRESCRITO

sobreescrever serve para fazer coisas do tipo como x2+3
TAG SUP

TEXTO SUBSCRITO

SUBSESCREVER serve para coisas como H2O
TAG SUB
